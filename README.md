# Проект Mesto

Сервис для обмена фотографиями интересных мест. Реализован функционал добавления, удаления и лайка карточек, редактирования профиля и аватара с сохранением данных на сервере через API.

**Ссылка на проект:** https://egorikbro.github.io/mesto-ad/

## Функциональность
*   Редактирование профиля
*   Обновление аватара
*   Добавление новых карточек (мест)
*   Удаление своих карточек с подтверждением
*   Лайк карточек
*   Просмотр фотографий в полном размере
*   Статистика пользователей (по клику на логотип)
*   Валидация форм

## Технологии
*   HTML5, CSS3
*   JavaScript (ES6+)
*   Webpack / Vite
*   Axios
*   API Яндекс.Практикума

## Запуск проекта

1.  Установите зависимости:
    ```bash
    npm install
    ```
2.  Запустите проект в режиме разработки:
    ```bash
    npm run dev
    ```
3.  Соберите проект для продакшена:
    ```bash
    npm run build
    ```

## Публикация
Деплой настроен автоматически через GitHub Actions. При пуше в ветку `main` запускается сборка и публикация на GitHub Pages.
